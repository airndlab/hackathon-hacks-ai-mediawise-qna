# RAG

## Разработка

Установить python.

1. Создать виртуальное окружение (если еще не создано):
   ```shell
   python -m venv venv
   ```

2. Активировать виртуальное окружение:
    - На Linux/macOS:
   ```shell
   source venv/bin/activate
   ```
    - На Windows:
   ```shell
   venv\Scripts\activate
   ```

3. Установить зависимости:
   ```shell
   pip install -r requirements.txt
   ```
